# Logistics Operations Dashboard

## Overview

This project demonstrates a professional Logistics Operations Dashboard developed using Microsoft Power BI and Excel.

The dashboard provides insights into shipment performance, carrier efficiency, route analysis, delivery status, and logistics cost optimization.

---

## Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX
- Data Modeling

---

## Dashboard Features

- Executive KPI Dashboard
- Shipment Tracking
- Delivery Performance
- Carrier Analysis
- Route Analysis
- Logistics Cost Analysis
- Interactive Filters
- Drill Through Reports

---

## KPIs

- Total Shipments
- Delivered Shipments
- Delayed Shipments
- On-Time Delivery %
- Average Transit Time
- Average Shipping Cost
- Cost per Shipment
